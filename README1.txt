1. Firstly the video in need is imported and each of its frames are extracted.
2. The frames are stored in a different folder and are named accordingly.
3. Then the difference between these frames are calculated and stored in a different folder.
4. Then the image is written in a form of RGB matrix.
5. The RGB matrix for first image is written completely, and for the rest RGB matrix of difference frame is stored.
6. These are written in a notepad file and later stacked and 7zipped to produce the compressed video
